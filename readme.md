## 安装

```shell
npm i -S gaea-injection-editor
```

## 组件说明

基于 [gaea-editor](https://github.com/ascoders/gaea-editor) 精简而成的纯 UI 编辑器。

* 移除了原编辑器的中组件的回调事件配置
* 可配置默认编辑器布局
